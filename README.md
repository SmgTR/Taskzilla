## Installation
- Use ```yarn install``` to install all dependencies.
- Use ``` yarn db-dev-seed``` to seed data.
- Use ```yarn dev``` to start project


Requries: PosgreSQL and env configuration.

```env
.env
DATABASE_URL=""
NEXTAUTH_URL = ""
NEXTAUTH_SECRET= ""
GOOGLE_CLIENT_ID = ""
GOOGLE_CLIENT_SECRET = ""
FACEBOOK_CLIENT_SECRET =""
FACEBOOK_CLIENT_ID = ""
```
